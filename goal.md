# Goal

## Project
cellprofiler — a python project.

## Description
CellProfiler is an image analysis software designed for biologists to quantitatively measure phenotypes from thousands of images automatically. The project provides a library of image processing functions, segmentation algorithms, measurement tools, and a pipeline-based workflow engine for automated image analysis in cell biology research.

## Scope
- 15-20 production source files to implement
- 10-15 test files to write
- Reproduce core image processing, segmentation, and measurement functionality
- Focus on the standalone library components (cellprofiler_library) and core engine (cellprofiler_core)
- Skip GUI-specific code (wxPython frontend)
